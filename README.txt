------------Simple Triangle Drawing Application------------
Written using Plain Javascript and HTML5. No third party libraries are used. Hence, no installation is necessary.
Copy of code is uploaded to private gitalab repository with timestamps for every progress. 
To request access, mail me at priyankamiranda2@gmail.com
How to use-
* Unzip and save folder to local system.
* Type in the complete location of the folder/App.html to your preferred browser
eg- file:///home/your/locationn/in/the/system/Draw_Anything/App.html
* Drawing a triangle. Click at two distinct points on the canvas.
* Draging triangle. Click the triangle then the new location.
* Deleting a triangle. Double click the triangle.
* Clearing screen. Doublel click all triangles or click clear button below.
